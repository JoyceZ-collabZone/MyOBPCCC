Westpac Open Banking Product API

This is the website for Westpac product information.

The Westpac Open Banking Product API enables third parties to retrieve information about products that Westpac currently offers. It is designed and documented to support the Australian Consumer Data Standards.

Learn more

Our Product API covers the following products for both retail and business customers:

  • Transaction Accounts
  • Savings Accounts
  • Term Deposits
  • Credit Cards
